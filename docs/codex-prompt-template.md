# Codex prompt template

Use this when starting work in this repo.

## Task
Describe the exact change.

## Constraints
- Use repo commands only: `make verify`, `make build`, `make run`, `make clean`.
- Keep automation in `Makefile` and `scripts/`.
- Do not edit generated files in `build/`.
- Keep the change small.

## Files to inspect first
List files.

## Verify
State which commands Codex must run.

## Output
- Brief summary
- Files changed
- Commands run
- Any blockers
